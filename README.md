# Mood to Telegram Emoji Bot 🤖😊

This project uses real-time facial expression detection to analyze a user's mood (such as happy, sad, or neutral) using a webcam. Once the mood is identified, the system sends the mood along with a corresponding emoji image to a Telegram bot.

## 🔧 Features
- Real-time mood detection using OpenCV and facial landmarks.
- Emoji mapping based on detected mood.
- Automatic message and emoji sending to Telegram bot.
- Lightweight and easy to run on any machine with a webcam.

## 🛠️ Technologies Used
- Python
- OpenCV
- Haar Cascade / DNN (for face detection)
- Telegram Bot API
- NumPy / TensorFlow
